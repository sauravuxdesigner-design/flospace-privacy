# FloSpace Privacy Policy

**Effective Date:** June 30, 2026

FloSpace is a Chrome extension that helps users save, organize, copy, and quickly access professional links while completing job applications and other professional application forms.

This Privacy Policy explains what information FloSpace stores, processes, and shares when you use the extension.

---

## Information Stored Locally

FloSpace stores your saved links locally using Chrome's `chrome.storage.local` API.

Saved data may include:

- Link label

- Link URL

- Link type (such as LinkedIn, GitHub, Portfolio, Resume, Website, or Custom)

- A local identifier used by the extension

Your saved links remain stored locally on your device. FloSpace does not operate a backend server to store your saved links or personal application data.

---

## Website Processing

FloSpace analyzes visible page elements, including form labels, placeholders, headings, and input fields, to detect professional application forms and improve the extension experience.

This processing occurs entirely within your browser.

FloSpace does **not** transmit page content, application answers, resume contents, clipboard contents, or your saved links to FloSpace servers.

---

## Chrome Permissions

### Storage

Used to securely store your saved links and preferences locally on your device.

### Website Access

FloSpace requires website access because professional application forms exist across many company career portals, applicant tracking systems, and hiring platforms.

Website access is used only to:

- Display the FloSpace floating button and interface

- Detect professional application pages

- Detect relevant form fields

- Reorder saved links based on the current application form

---

## Analytics

FloSpace uses **PostHog** to collect limited anonymous usage analytics to improve the extension.

Examples of analytics events include:

- `extension_opened`

- `link_added`

- `link_copied`

- `search_used`

- `fab_dragged`

For events such as `link_added` and `link_copied`, FloSpace sends only the general link type (for example: LinkedIn, GitHub, Portfolio, Resume, or Other).

FloSpace does **not** send:

- Saved link URLs

- Search text

- Clipboard contents

- Email addresses

- Names

- Resume contents

- Portfolio contents

- Page URLs

- Browsing history

- Application form answers

FloSpace disables PostHog automatic page tracking, autocapture, session recording, surveys, heatmaps, page leave tracking, and user identification.

---

## Third-Party Services

### PostHog

FloSpace uses PostHog solely to receive the anonymous analytics events described above.

### Google Favicon Service

To display website icons for saved links, FloSpace may request favicons from Google's public favicon service.

These requests are used only to retrieve website icons for display within the extension.

### Ko-fi

If you choose to support FloSpace, clicking the **"Buy Me a Tea"** button opens Ko-fi in a new browser tab.

---

## Clipboard

When you click the **Copy** button, FloSpace copies the selected saved link to your clipboard using the browser Clipboard API.

FloSpace does **not** read, monitor, or upload clipboard contents.

---

## Data Sharing

FloSpace does **not** sell personal information.

FloSpace does **not** use your data for advertising, creditworthiness, lending decisions, or unrelated purposes.

Limited anonymous analytics events are processed by PostHog.

---

## Data Security

FloSpace is designed to minimize data collection.

Saved links are stored locally within your Chrome profile using Chrome Extension Storage.

FloSpace does not store your saved links or professional application data on its own servers.

---

## Data Retention

Saved links remain stored locally until you:

- Delete them within FloSpace

- Clear Chrome extension data

- Reset your browser profile

- Uninstall the extension

Analytics events are retained by PostHog according to the PostHog project's retention settings.

---

## Your Choices

You may:

- Add, edit, or delete saved links at any time.

- Remove all locally stored FloSpace data by uninstalling the extension or clearing Chrome extension storage.

---

## Children's Privacy

FloSpace is not intended for children under the age of 13 and does not knowingly collect personal information from children.
